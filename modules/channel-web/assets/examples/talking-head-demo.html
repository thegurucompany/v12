Chato 3D Avatar Demo

Prueba del avatar 3D TalkingHead con lip-sync y ElevenLabs voice agent.

Requisitos:
1. El bot debe tener voiceAgent.enabled: true con agentId y apiKey configurados
2. Agregar a la config del bot: avatar3dEnabled: true y avatar3dModelUrl
3. Necesitas un modelo .glb con Oculus viseme blend shapes (ver README)
4. Puedes crear un avatar de prueba en avaturn.me
